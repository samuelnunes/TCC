O VisEdu-MAT é um aplicativo que visa auxiliar o ensino de funções matemáticas para os alunos do curso
de Bacharelado em Ciência da Computação.

1 - Padrão de notação do VisEdu-MAT

O VisEdu-Mat usa o padrão de notação do ASCIIMathML para exibir uma função. Portanto deve-se seguir algumas convenções
impostas pela biblioteca, que são:

Para operadores aritméticos:

adição = + (Ex: 3 + x)
subtração = - (Ex: 3 - x)
divisão = / (Ex: 3 / x)
multiplicação = * (Ex: 3 * x)
potenciação = ^ (Ex: 3 ^ x)
radiciação = sqrt(x) (Ex: sqrt(3))
logaritmo = log(x) (Ex: log(3))

Para funções trigonométricas:

seno = sin(x)
cosseno = cos(x)
tangente = tan(x)

Para funções trigonométricas inversas:

arco seno = asin(x)
arco cosseno = acos(x)
arco tangente = atan(x)

3 - Alterando o contradomínio de uma Função

Uma função liga um domínio (conjunto de valores de entrada) com um contradomínio (conjunto de valores de saída)
de tal forma, que cada elemento do domínio está associado exatamente a um elemento do contradomínio.
Isso quer dizer que:

- Quando o contradomínio for 'z', a função digitada poderá ser sobre o domínio de 'x' e/ou 'y' (Ex: 2*x + 3*y).
- Quando o contradomínio for 'x', a função digitada poderá ser sobre o domínio de 'z' e/ou 'y' (Ex: 2*z - 2*y).
- Quando o contradomínio for 'y', a função digitada poderá ser sobre o domínio de 'x' e/ou 'z' (Ex: 4*x * 3*z).
